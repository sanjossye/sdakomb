1. Dasar Pointer

Pointer adalah variabel yang menyimpan alamat memori dari variabel lain. Operator & digunakan untuk mengambil alamat, sedangkan operator * digunakan untuk mengakses nilai di alamat tersebut (dereference).

Kode Program

C++ 
#include <iostream>
using namespace std;

int main()
{
    int a = 10;
    int *b = &a;  // b menyimpan alamat memori a
    *b = 20;      // mengubah nilai a melalui pointer b

    cout << "mem address of a -> " << &a << endl;
    cout << "nilai a adalah   -> " << a  << endl;

    cout << "mem address of b -> " << &b << endl;
    cout << "nilai b adalah   -> " << *b << endl;

    return 0;
}

Penjelasan Program

Poin Penting:
  • int *b = &a → pointer b menunjuk ke alamat variabel a
  • *b = 20 → mengubah nilai di alamat yang ditunjuk, sehingga nilai a ikut berubah menjadi 20
  • &b adalah alamat dari variabel pointer itu sendiri, berbeda dari &a
  • Karena b menunjuk ke a, nilai *b dan nilai a selalu sama

Contoh Output

mem address of a -> 0x7ffe4abc1230
nilai a adalah   -> 20
mem address of b -> 0x7ffe4abc1238
nilai b adalah   -> 20

2. Swap Manual (di main)

Pertukaran nilai dua variabel secara langsung di dalam main() menggunakan variabel sementara temp. Berhasil karena kita langsung menyentuh variabel aslinya.

Kode Program

C++ 
#include <iostream>
using namespace std;

int main()
{
    int a = 10;
    int b = 20;

    int temp;
    temp = a;   // temp = 10
    a = b;      // a = 20
    b = temp;   // b = 10

    cout << "mem address of a -> " << &a << endl;
    cout << "nilai a adalah   -> " << a  << endl;

    cout << "mem address of b -> " << &b << endl;
    cout << "nilai b adalah   -> " << b  << endl;
}

Penjelasan Program

Swap berhasil karena dilakukan langsung di scope yang sama dengan variabel a dan b. Variabel temp berfungsi sebagai tempat penyimpanan sementara agar nilai tidak hilang saat pertukaran.

Contoh Output

mem address of a -> 0x7ffe...
nilai a adalah   -> 20
mem address of b -> 0x7ffe...
nilai b adalah   -> 10

3. Pass by Value (Swap Gagal)

Fungsi tukar(int a, int b) menerima salinan nilai, bukan variabel aslinya. Pertukaran terjadi di dalam fungsi tetapi tidak mempengaruhi variabel di main().

Kode Program

C++ 
#include <iostream>
using namespace std;

void tukar(int a, int b)    // a dan b adalah SALINAN nilai
{
    int temp;
    temp = a;   // perubahan hanya berlaku di sini
    a = b;
    b = temp;
}

int main()
{
    int a = 10;
    int b = 20;

    tukar(a, b);   // mengirim NILAI, bukan alamat

    cout << "mem address of a -> " << &a << endl;
    cout << "nilai a adalah   -> " << a  << endl;  // tetap 10

    cout << "mem address of b -> " << &b << endl;
    cout << "nilai b adalah   -> " << b  << endl;  // tetap 20

    return 0;
}

Penjelasan Program

⚠ Mengapa Gagal?
  • Parameter fungsi a dan b adalah variabel baru yang hanya menyimpan salinan nilai dari main()
  • Pertukaran di dalam fungsi tidak menyentuh memori variabel asli di main()
  • Saat fungsi selesai dieksekusi, variabel salinan tersebut langsung dihapus dari stack

Contoh Output

mem address of a -> 0x7ffe...
nilai a adalah   -> 10
mem address of b -> 0x7ffe...
nilai b adalah   -> 20

4. Pass by Pointer (Swap Berhasil)

Fungsi tukar(int *a, int *b) menerima alamat memori variabel aslinya. Dengan operator *, fungsi dapat langsung memanipulasi nilai di alamat tersebut.

Kode Program

C++ 
#include <iostream>
using namespace std;

void tukar(int *a, int *b)   // a dan b adalah POINTER ke variabel asli
{
    int temp;
    temp = *a;   // ambil nilai di alamat a
    *a = *b;     // tulis nilai b ke alamat a
    *b = temp;   // tulis temp ke alamat b
}

int main()
{
    int a = 10;
    int b = 20;

    tukar(&a, &b);   // mengirim ALAMAT variabel, bukan nilainya

    cout << "mem address of a -> " << &a << endl;
    cout << "nilai a adalah   -> " << a  << endl;  // 20

    cout << "mem address of b -> " << &b << endl;
    cout << "nilai b adalah   -> " << b  << endl;  // 10

    return 0;
}

Penjelasan Program

✅ Mengapa Berhasil?
  • tukar(&a, &b) mengirimkan alamat memori nyata dari variabel a dan b
  • Di dalam fungsi, *a dan *b langsung mengakses lokasi memori yang sama dengan variabel di main()
  • Perubahan yang dilakukan di dalam fungsi langsung terlihat di main()

Contoh Output

mem address of a -> 0x7ffe...
nilai a adalah   -> 20
mem address of b -> 0x7ffe...
nilai b adalah   -> 10

5. Pointer ke Pointer (Double Pointer)

Double pointer adalah pointer yang menyimpan alamat dari pointer lain. Dideklarasikan dengan dua tanda bintang (**), dan memungkinkan kita mengakses atau mengubah nilai variabel melalui dua tingkat dereferensi.

Kode Program

C++ 
#include <iostream>
using namespace std;

int main()
{
    int x = 99;
    int *ptr = &x;      // ptr menyimpan alamat x
    int **pptr = &ptr;  // pptr menyimpan alamat ptr

    cout << "x      = " << x      << endl;
    cout << "*ptr   = " << *ptr   << endl;
    cout << "**pptr = " << **pptr << endl;

    // ubah nilai x lewat pointer ke pointer
    **pptr = 150;

    cout << "x setelah diubah lewat **pptr = " << x << endl;

    return 0;
}

Penjelasan Program

Rantai Memori:
  • x → menyimpan nilai 99, berada di suatu alamat memori, misal 0x1000
  • ptr → menyimpan alamat x (0x1000), sehingga *ptr == 99
  • pptr → menyimpan alamat ptr, sehingga *pptr == &x dan **pptr == 99
Cara Membaca Dereferensi:
  • *ptr → "pergi ke alamat yang disimpan ptr, ambil nilainya" → 99
  • *pptr → "pergi ke alamat yang disimpan pptr" → menghasilkan ptr (sebuah pointer)
  • **pptr → dereferensi dua kali → sama dengan *ptr → sama dengan x
  • **pptr = 150 → langsung mengubah nilai x melalui dua tingkat pointer

Visualisasi Memori

[ x    | 99  ] ← alamat 0x1000
[ ptr  | 0x1000 ] ← alamat 0x1008   →  *ptr  == 99
[ pptr | 0x1008 ] ← alamat 0x1010   →  **pptr == 99

Contoh Output

x      = 99
*ptr   = 99
**pptr = 99
x setelah diubah lewat **pptr = 150

6. Dasar Struct

Struct (structure) adalah tipe data bentukan yang menggabungkan beberapa variabel dengan tipe yang berbeda-beda ke dalam satu kelompok. Struct sangat berguna untuk merepresentasikan sebuah entitas nyata, misalnya data saham di aplikasi seperti Stockbit.

Sintaks Dasar

C++ 
struct NamaStruct {
    tipe_data anggota1;
    tipe_data anggota2;
    // ...
};

Kode Program

C++ 
#include <iostream>
using namespace std;

struct Saham {
    string kode;
    string emiten;
    int harga;
    string pemilik;
    string kategori;
};

int main() {

    // Banyak saham seperti di aplikasi Stockbit
    Saham daftarSaham[4] = {
        {"BBCA", "Bank Central Asia", 7900, "Hartono Family", "Perbankan"},
        {"RLCO", "Abadi Lestari Indonesia", 7700, "Edwin Pranata", "Konsumen non-primer"},
        {"BUMI", "Bumi Resources", 250, "Mach Energy", "Pertambangan"},
        {"DEWA", "Darma Henwa", 505, "Zurich Asset International", "Pertambangan"}
    };

    // Menampilkan semua saham
    for (int i = 0; i < 4; i++) {
        cout << "===== Saham ke-" << i+1 << " =====" << endl;
        cout << "Kode     : " << daftarSaham[i].kode << endl;
        cout << "Emiten   : " << daftarSaham[i].emiten << endl;
        cout << "Harga    : Rp." << daftarSaham[i].harga << endl;
        cout << "Pemilik  : " << daftarSaham[i].pemilik << endl;
        cout << "Kategori : " << daftarSaham[i].kategori << endl;
        cout << endl;
    }

    return 0;
}

Penjelasan Program

Poin Penting:
  • struct Saham { ... } → mendefinisikan blueprint/cetakan tipe data baru bernama Saham
  • Setiap variabel di dalam struct disebut member atau anggota
  • Saham daftarSaham[4] → membuat array berisi 4 objek bertipe Saham
  • Akses member menggunakan operator titik → daftarSaham[i].kode, daftarSaham[i].harga, dst.
  • Inisialisasi dilakukan dengan kurung kurawal {} sesuai urutan member dalam struct
💡 Analogi Struct:

Bayangkan struct seperti formulir isian. Struct Saham adalah template formulirnya, sedangkan setiap objek daftarSaham[i] adalah satu lembar formulir yang sudah diisi dengan data saham tertentu.

7. Struct: Pass by Value

Ketika struct dioper ke fungsi secara pass by value, fungsi menerima salinan penuh dari seluruh member struct. Perubahan apapun di dalam fungsi tidak akan mempengaruhi data asli di main().

Kode Program

C++ 
// Tampilkan struct (pass by value — cukup untuk baca-baca)
void tampilkanSaham_ByValue(Saham s) {
    cout << "\n[PASS BY VALUE]" << endl;
    cout << "Kode     : " << s.kode << endl;
    cout << "Emiten   : " << s.emiten << endl;
    cout << "Harga    : Rp." << s.harga << endl;
    cout << "Pemilik  : " << s.pemilik << endl;
    cout << "Kategori : " << s.kategori << endl;
}

// Coba ubah harga — TIDAK AKAN BERPENGARUH ke data asli!
void naikkanHarga_ByValue(Saham s, int kenaikan) {
    s.harga += kenaikan;
    cout << "\n[BY VALUE] Harga di dalam fungsi: Rp." << s.harga << endl;
}

int main() {
    Saham daftarSaham[4] = { ... };

    tampilkanSaham_ByValue(daftarSaham[0]);

    cout << "Harga SEBELUM naikkan (di main): Rp." << daftarSaham[0].harga << endl;
    naikkanHarga_ByValue(daftarSaham[0], 100);
    cout << "Harga SETELAH naikkan (di main): Rp." << daftarSaham[0].harga << endl;
    cout << "❌ HARGA TIDAK BERUBAH! (karena by value, cuma copy)" << endl;
}

Penjelasan Program

⚠ Mengapa Tidak Berubah?
  • Parameter Saham s di fungsi adalah salinan baru dari struct asli
  • s.harga += kenaikan hanya mengubah salinan, bukan struct di main()
  • Begitu fungsi selesai, salinan tersebut langsung dihapus dari stack
  • Pass by value cocok digunakan jika fungsi hanya membaca data, bukan mengubahnya

Contoh Output

[PASS BY VALUE]
Kode     : BBCA
Emiten   : Bank Central Asia
Harga    : Rp.7900
Pemilik  : Hartono Family
Kategori : Perbankan

Harga SEBELUM naikkan (di main): Rp.7900
[BY VALUE] Harga di dalam fungsi: Rp.8000
Harga SETELAH naikkan (di main): Rp.7900
❌ HARGA TIDAK BERUBAH! (karena by value, cuma copy)

8. Struct: Pass by Reference

Ketika struct dioper secara pass by reference (Saham &s), fungsi mendapatkan referensi langsung ke struct asli. Perubahan di dalam fungsi akan langsung mempengaruhi data aslinya.

Kode Program

C++ 
// Tampilkan struct (pass by reference — lebih efisien karena tidak copy)
void tampilkanSaham_ByReference(Saham &s) {
    cout << "\n[PASS BY REFERENCE]" << endl;
    cout << "Kode     : " << s.kode << endl;
    cout << "Emiten   : " << s.emiten << endl;
    cout << "Harga    : Rp." << s.harga << endl;
    cout << "Pemilik  : " << s.pemilik << endl;
    cout << "Kategori : " << s.kategori << endl;
}

// Ubah harga — AKAN BERPENGARUH ke data asli!
void naikkanHarga_ByReference(Saham &s, int kenaikan) {
    s.harga += kenaikan;
    cout << "\n[BY REFERENCE] Harga di dalam fungsi: Rp." << s.harga << endl;
}

int main() {
    Saham daftarSaham[4] = { ... };

    tampilkanSaham_ByReference(daftarSaham[1]);

    cout << "Harga SEBELUM naikkan (di main): Rp." << daftarSaham[1].harga << endl;
    naikkanHarga_ByReference(daftarSaham[1], 100);
    cout << "Harga SETELAH naikkan (di main): Rp." << daftarSaham[1].harga << endl;
    cout << "✅ HARGA BERUBAH! (karena by reference, langsung ke aslinya)" << endl;
}

Penjelasan Program

✅ Mengapa Berhasil?
  • Operator & pada parameter membuat s menjadi alias dari struct asli
  • s.harga += kenaikan langsung memodifikasi struct yang ada di main()
  • Pass by reference juga lebih efisien memori dari pass by value karena tidak membuat salinan
  • Akses member tetap menggunakan operator titik (.), bukan panah (->)

Contoh Output

[PASS BY REFERENCE]
Kode     : RLCO
Emiten   : Abadi Lestari Indonesia
Harga    : Rp.7700
Pemilik  : Edwin Pranata
Kategori : Konsumen non-primer

Harga SEBELUM naikkan (di main): Rp.7700
[BY REFERENCE] Harga di dalam fungsi: Rp.7800
Harga SETELAH naikkan (di main): Rp.7800
✅ HARGA BERUBAH! (karena by reference, langsung ke aslinya)

9. Struct: Pass by Pointer

Ketika struct dioper secara pass by pointer (Saham *s), fungsi menerima alamat memori struct asli. Akses member dilakukan menggunakan operator panah (->) sebagai pengganti (*s).member.

Kode Program

C++ 
// Tampilkan struct (pass by pointer — pakai -> untuk akses member)
void tampilkanSaham_ByPointer(Saham *s) {
    cout << "\n[PASS BY POINTER]" << endl;
    cout << "Kode     : " << s->kode << endl;      // Pakai -> untuk pointer
    cout << "Emiten   : " << s->emiten << endl;
    cout << "Harga    : Rp." << s->harga << endl;
    cout << "Pemilik  : " << s->pemilik << endl;
    cout << "Kategori : " << s->kategori << endl;
}

// Ubah harga — AKAN BERPENGARUH ke data asli!
void naikkanHarga_ByPointer(Saham *s, int kenaikan) {
    s->harga += kenaikan;
    cout << "\n[BY POINTER] Harga di dalam fungsi: Rp." << s->harga << endl;
}

int main() {
    Saham daftarSaham[4] = { ... };

    tampilkanSaham_ByPointer(&daftarSaham[2]);   // Kirim alamat pakai &

    cout << "Harga SEBELUM naikkan (di main): Rp." << daftarSaham[2].harga << endl;
    naikkanHarga_ByPointer(&daftarSaham[2], 50);
    cout << "Harga SETELAH naikkan (di main): Rp." << daftarSaham[2].harga << endl;
    cout << "✅ HARGA BERUBAH! (karena by pointer, akses langsung via alamat)" << endl;
}

Penjelasan Program

✅ Poin Penting Pass by Pointer:
  • Pemanggilan menggunakan &daftarSaham[2] → mengirim alamat memori struct, bukan nilainya
  • Di dalam fungsi, s adalah pointer ke struct asli
  • Operator -> adalah shorthand dari (*s). → dereference lalu akses member
  • s->harga sama persis dengan (*s).harga
💡 Operator Titik vs Panah:
  • s.harga → digunakan jika s adalah struct biasa atau reference
  • s->harga → digunakan jika s adalah pointer ke struct

Contoh Output

[PASS BY POINTER]
Kode     : BUMI
Emiten   : Bumi Resources
Harga    : Rp.250
Pemilik  : Mach Energy
Kategori : Pertambangan

Harga SEBELUM naikkan (di main): Rp.250
[BY POINTER] Harga di dalam fungsi: Rp.300
Harga SETELAH naikkan (di main): Rp.300
✅ HARGA BERUBAH! (karena by pointer, akses langsung via alamat)

10. Array Struct ke Fungsi

Array of struct dapat dioper ke fungsi menggunakan pointer. Saat nama array ditulis tanpa kurung siku (daftarSaham), C++ secara otomatis mengubahnya menjadi pointer ke elemen pertama array. Itulah mengapa parameter fungsinya bertipe Saham *daftar.

Kode Program

C++ 
void tampilkanSemuaSaham(Saham *daftar, int ukuran) {
    cout << "\n===== MENAMPILKAN SEMUA SAHAM =====" << endl;
    for (int i = 0; i < ukuran; i++) {
        cout << "\n----- Saham ke-" << i+1 << " -----" << endl;
        cout << "Kode     : " << daftar[i].kode << endl;
        cout << "Emiten   : " << daftar[i].emiten << endl;
        cout << "Harga    : Rp." << daftar[i].harga << endl;
        cout << "Pemilik  : " << daftar[i].pemilik << endl;
        cout << "Kategori : " << daftar[i].kategori << endl;
    }
}

int main() {
    Saham daftarSaham[4] = { ... };

    // nama array otomatis jadi pointer ke elemen pertama
    tampilkanSemuaSaham(daftarSaham, 4);
}

Penjelasan Program

Poin Penting:
  • daftarSaham saat dioper ke fungsi secara otomatis menjadi &daftarSaham[0] (pointer ke elemen pertama)
  • Parameter int ukuran wajib dioper karena pointer tidak menyimpan informasi panjang array
  • Di dalam fungsi, daftar[i] tetap menggunakan notasi indeks seperti biasa meskipun daftar adalah pointer
  • Ini adalah cara standar C++ untuk mengoper array ke dalam fungsi

Contoh Output

===== MENAMPILKAN SEMUA SAHAM =====

----- Saham ke-1 -----
Kode     : BBCA
Emiten   : Bank Central Asia
Harga    : Rp.7900
...

----- Saham ke-4 -----
Kode     : DEWA
Emiten   : Darma Henwa
Harga    : Rp.505
Pemilik  : Zurich Asset International
Kategori : Pertambangan

11. Fungsi yang Mengembalikan Pointer Struct

Fungsi dapat mengembalikan pointer ke struct (Saham*). Ini berguna untuk mencari data dalam array — jika ditemukan, kembalikan alamat memori elemen tersebut; jika tidak, kembalikan nullptr.

Kode Program

C++ 
Saham* cariSaham(Saham *daftar, int ukuran, string kode) {
    for (int i = 0; i < ukuran; i++) {
        if (daftar[i].kode == kode) {
            return &daftar[i];   // kembalikan alamat elemen yang cocok
        }
    }
    return nullptr;   // tidak ditemukan
}

int main() {
    Saham daftarSaham[4] = { ... };

    string kodeYangDicari = "BUMI";
    Saham* hasil = cariSaham(daftarSaham, 4, kodeYangDicari);

    if (hasil != nullptr) {
        cout << "✅ Saham " << kodeYangDicari << " ditemukan!" << endl;
        cout << "Emiten : " << hasil->emiten << endl;
        cout << "Harga  : Rp." << hasil->harga << endl;
    } else {
        cout << "Saham tidak ditemukan!" << endl;
    }
}

Penjelasan Program

Poin Penting:
  • Return type Saham* → fungsi mengembalikan pointer ke struct, bukan salinannya
  • return &daftar[i] → mengembalikan alamat elemen array yang cocok
  • nullptr → nilai khusus yang berarti pointer tidak menunjuk ke mana-mana (tidak ditemukan)
  • Cek hasil != nullptr sebelum menggunakan pointer untuk menghindari crash
  • Hasil akses menggunakan -> karena hasil bertipe pointer
⚠ Hati-hati dengan nullptr:

Mengakses member dari pointer nullptr akan menyebabkan program crash (segmentation fault). Selalu cek != nullptr sebelum menggunakan pointer hasil kembalian fungsi pencarian.

Contoh Output

✅ Saham BUMI ditemukan!
Emiten : Bumi Resources
Harga  : Rp.300

    cout << "\n[BY REFERENCE] Harga di dalam fungsi: Rp." << s.harga << endl;
}

void tampilkanSemuaSaham(Saham *daftar, int ukuran) {
    cout << "\n===== MENAMPILKAN SEMUA SAHAM =====" << endl;
    for (int i = 0; i < ukuran; i++) {
        cout << "\n----- Saham ke-" << i+1 << " -----" << endl;
        cout << "Kode     : " << daftar[i].kode << endl;
        cout << "Emiten   : " << daftar[i].emiten << endl;
        cout << "Harga    : Rp." << daftar[i].harga << endl;
        cout << "Pemilik  : " << daftar[i].pemilik << endl;
        cout << "Kategori : " << daftar[i].kategori << endl;
    }
}


Saham* cariSaham(Saham *daftar, int ukuran, string kode) {
    for (int i = 0; i < ukuran; i++) {
        if (daftar[i].kode == kode) {
            return &daftar[i];
        }
    }
    return nullptr;
}

Kesimpulan

Memahami pointer dan struct adalah kunci untuk mengerti bagaimana data dikelola di memori dalam C++. Berikut rangkuman yang telah dipelajari:

Poin-Poin Penting:
  • Pointer: Variabel yang menyimpan alamat memori variabel lain
  • Pass by Value: Fungsi menerima salinan — perubahan tidak mempengaruhi variabel asli
  • Pass by Pointer: Fungsi menerima alamat — perubahan langsung ke variabel asli
  • Double Pointer: Pointer yang menyimpan alamat pointer lain — akses nilai melalui dua tingkat dereferensi (**pptr)
  • Struct: Tipe data bentukan yang mengelompokkan beberapa variabel berbeda tipe menjadi satu entitas
  • Struct + Pass by Value: Salinan struct dioper — perubahan tidak mempengaruhi data asli
  • Struct + Pass by Reference (&): Alias langsung ke struct asli — lebih efisien dan perubahan langsung terasa
  • Struct + Pass by Pointer (*): Alamat struct dioper — akses member menggunakan ->
  • Array Struct ke Fungsi: Nama array otomatis menjadi pointer ke elemen pertama
  • Fungsi Kembalikan Pointer: Gunakan nullptr untuk kondisi tidak ditemukan, selalu cek sebelum digunakan
