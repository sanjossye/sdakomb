1. Dasar Pointer

Pointer adalah variabel yang menyimpan alamat memori dari variabel lain. Operator & digunakan untuk mengambil alamat, sedangkan operator * digunakan untuk mengakses nilai di alamat tersebut (dereference).

Kode Program

C++ 
#include <iostream>
using namespace std;

int main()
{
    int a = 10;
    int *b = &a;  // b menyimpan alamat memori a
    *b = 20;      // mengubah nilai a melalui pointer b

    cout << "mem address of a -> " << &a << endl;
    cout << "nilai a adalah   -> " << a  << endl;

    cout << "mem address of b -> " << &b << endl;
    cout << "nilai b adalah   -> " << *b << endl;

    return 0;
}

Penjelasan Program

Poin Penting:
  • int *b = &a → pointer b menunjuk ke alamat variabel a
  • *b = 20 → mengubah nilai di alamat yang ditunjuk, sehingga nilai a ikut berubah menjadi 20
  • &b adalah alamat dari variabel pointer itu sendiri, berbeda dari &a
  • Karena b menunjuk ke a, nilai *b dan nilai a selalu sama

Contoh Output

mem address of a -> 0x7ffe4abc1230
nilai a adalah   -> 20
mem address of b -> 0x7ffe4abc1238
nilai b adalah   -> 20

2. Swap Manual (di main)

Pertukaran nilai dua variabel secara langsung di dalam main() menggunakan variabel sementara temp. Berhasil karena kita langsung menyentuh variabel aslinya.

Kode Program

C++ 
#include <iostream>
using namespace std;

int main()
{
    int a = 10;
    int b = 20;

    int temp;
    temp = a;   // temp = 10
    a = b;      // a = 20
    b = temp;   // b = 10

    cout << "mem address of a -> " << &a << endl;
    cout << "nilai a adalah   -> " << a  << endl;

    cout << "mem address of b -> " << &b << endl;
    cout << "nilai b adalah   -> " << b  << endl;
}

Penjelasan Program

Swap berhasil karena dilakukan langsung di scope yang sama dengan variabel a dan b. Variabel temp berfungsi sebagai tempat penyimpanan sementara agar nilai tidak hilang saat pertukaran.

Contoh Output

mem address of a -> 0x7ffe...
nilai a adalah   -> 20
mem address of b -> 0x7ffe...
nilai b adalah   -> 10

3. Pass by Value (Swap Gagal)

Fungsi tukar(int a, int b) menerima salinan nilai, bukan variabel aslinya. Pertukaran terjadi di dalam fungsi tetapi tidak mempengaruhi variabel di main().

Kode Program

C++ 
#include <iostream>
using namespace std;

void tukar(int a, int b)    // a dan b adalah SALINAN nilai
{
    int temp;
    temp = a;   // perubahan hanya berlaku di sini
    a = b;
    b = temp;
}

int main()
{
    int a = 10;
    int b = 20;

    tukar(a, b);   // mengirim NILAI, bukan alamat

    cout << "mem address of a -> " << &a << endl;
    cout << "nilai a adalah   -> " << a  << endl;  // tetap 10

    cout << "mem address of b -> " << &b << endl;
    cout << "nilai b adalah   -> " << b  << endl;  // tetap 20

    return 0;
}

Penjelasan Program

⚠ Mengapa Gagal?
  • Parameter fungsi a dan b adalah variabel baru yang hanya menyimpan salinan nilai dari main()
  • Pertukaran di dalam fungsi tidak menyentuh memori variabel asli di main()
  • Saat fungsi selesai dieksekusi, variabel salinan tersebut langsung dihapus dari stack

Contoh Output

mem address of a -> 0x7ffe...
nilai a adalah   -> 10
mem address of b -> 0x7ffe...
nilai b adalah   -> 20

4. Pass by Pointer (Swap Berhasil)

Fungsi tukar(int *a, int *b) menerima alamat memori variabel aslinya. Dengan operator *, fungsi dapat langsung memanipulasi nilai di alamat tersebut.

Kode Program

C++ 
#include <iostream>
using namespace std;

void tukar(int *a, int *b)   // a dan b adalah POINTER ke variabel asli
{
    int temp;
    temp = *a;   // ambil nilai di alamat a
    *a = *b;     // tulis nilai b ke alamat a
    *b = temp;   // tulis temp ke alamat b
}

int main()
{
    int a = 10;
    int b = 20;

    tukar(&a, &b);   // mengirim ALAMAT variabel, bukan nilainya

    cout << "mem address of a -> " << &a << endl;
    cout << "nilai a adalah   -> " << a  << endl;  // 20

    cout << "mem address of b -> " << &b << endl;
    cout << "nilai b adalah   -> " << b  << endl;  // 10

    return 0;
}

Penjelasan Program

✅ Mengapa Berhasil?
  • tukar(&a, &b) mengirimkan alamat memori nyata dari variabel a dan b
  • Di dalam fungsi, *a dan *b langsung mengakses lokasi memori yang sama dengan variabel di main()
  • Perubahan yang dilakukan di dalam fungsi langsung terlihat di main()

Contoh Output

mem address of a -> 0x7ffe...
nilai a adalah   -> 20
mem address of b -> 0x7ffe...
nilai b adalah   -> 10

Kesimpulan

Memahami pointer adalah kunci untuk mengerti bagaimana data dikelola di memori dalam C++. Berikut rangkuman yang telah dipelajari:

Poin-Poin Penting:
  • Pointer: Variabel yang menyimpan alamat memori variabel lain
  • Pass by Value: Fungsi menerima salinan — perubahan tidak mempengaruhi variabel asli
  • Pass by Pointer: Fungsi menerima alamat — perubahan langsung ke variabel asli
  • Tips: C++ juga punya pass by reference (int &a) sebagai alternatif yang lebih bersih dari pointer
