Pengantar — Tiga Jenis Linked List

Linked List adalah struktur data di mana setiap elemen (node) menyimpan data sekaligus pointer ke node lain. Ada tiga varian utama yang dipelajari di sini:

Perbandingan Tiga Jenis Linked List:
  • Single Linked List (SLL) → setiap node hanya punya pointer next; traversal satu arah; ujung menunjuk NULL
  • Double Linked List (DLL) → setiap node punya prev dan next; traversal dua arah; ujung menunjuk NULL
  • Circular Single Linked List (CSLL) → seperti SLL tapi tail->next menunjuk kembali ke head; tidak ada NULL
Single LL:
NULL <- [A] -> [B] -> [C] -> NULL
        head           tail

Double LL:
NULL <- [A] <-> [B] <-> [C] -> NULL
        head             tail

Circular Single LL:
+-----------------------------------+
|                                   |
+-> [A] -> [B] -> [C] -> [D] ------+
    head            tail

Apa itu Single Linked List?

Single Linked List (SLL) adalah linked list paling dasar. Setiap node menyimpan data dan satu pointer next yang menunjuk ke node berikutnya. Node terakhir (tail) memiliki next = NULL sebagai penanda akhir list.

Karakteristik SLL:
  • Traversal hanya bisa ke depan (satu arah)
  • Insert/delete di awal: O(1)
  • Insert/delete di akhir: O(1) jika ada pointer tail; O(n) jika tidak
  • Insert/delete di tengah: O(n) — butuh traversal
  • Cocok untuk: stack, queue sederhana, undo list

Deklarasi Struct & Variabel Global

Struct Buku mendefinisikan bentuk setiap node. Variabel global digunakan agar semua fungsi bisa mengakses pointer list tanpa harus passing parameter terus-menerus.

Kode Program

C++ 
struct Buku {
    string judul, pengarang;
    int tahunTerbit;
    Buku *next;          // pointer ke node berikutnya; NULL jika node terakhir
};

Buku *head;     // node pertama
Buku *tail;     // node terakhir
Buku *cur;      // cursor traversal
Buku *newNode;  // node baru saat insert
Buku *del;      // node yang akan dihapus
Buku *before;   // node sebelum target (untuk removeMiddle)
Fungsi Tiap Variabel Global:
  • head → pintu masuk list; semua traversal dimulai dari sini
  • tail → memungkinkan addLast O(1) tanpa perlu traversal ke akhir
  • cur → pointer sementara untuk berjalan node demi node
  • newNode → menampung node baru sebelum disambungkan ke list
  • del → menyimpan node yang hendak dihapus sebelum memorinya dibebaskan
  • before → dipakai di removeMiddle untuk menyimpan node sebelum target

Create

Fungsi createSingleLinkedList() membuat node pertama sekaligus menginisialisasi head dan tail. Dipanggil hanya sekali di awal program.

Kode Program

C++ 
void createSingleLinkedList(string judul, string pengarang, int tB) {
    head = new Buku();
    head->judul       = judul;
    head->pengarang   = pengarang;
    head->tahunTerbit = tB;
    head->next        = NULL;   // node pertama belum punya penerus
    tail = head;                // satu-satunya node = head sekaligus tail
}
Alur Eksekusi Create:
  • new Buku() → alokasi memori di heap untuk node pertama
  • Isi semua member dengan nilai parameter yang diterima
  • head->next = NULL → tidak ada node setelahnya
  • tail = head → karena hanya ada satu node, head dan tail adalah node yang sama

Count

Fungsi countSingleLinkedList() menghitung jumlah node dalam list dengan cara traversal dari head ke tail.

Kode Program

C++ 
int countSingleLinkedList() {
    cur = head;
    int jumlah = 0;
    while (cur != NULL) {
        jumlah++;
        cur = cur->next;
    }
    return jumlah;
}
Poin Penting:
  • Mulai dari head, maju terus sampai cur == NULL
  • Setiap kali masuk iterasi, jumlah bertambah satu
  • Kompleksitas: O(n) — harus menelusuri seluruh list

Add First

Menambahkan node baru di posisi paling depan. Node baru menjadi head baru. Operasi ini O(1).

Kode Program

C++ 
void addFirst(string judul, string pengarang, int tB) {
    newNode = new Buku();
    newNode->judul       = judul;
    newNode->pengarang   = pengarang;
    newNode->tahunTerbit = tB;
    newNode->next = head;   // node baru menunjuk ke head lama
    head = newNode;         // head digeser ke node baru
}
⚠ Urutan Wajib Diperhatikan:
  • newNode->next = head harus dilakukan SEBELUM head = newNode
  • Jika head digeser duluan, referensi ke node lama hilang selamanya
  • Pointer tail tidak berubah — head berubah, tail tetap

Visualisasi

Sebelum: head --> [Kata] --> NULL
                   tail --> [Kata]

addFirst("Dia adalah Kakakku", ...)
newNode->next = head   --> [DiaAdlhKkk] --> [Kata]
head          = newNode

Sesudah: head --> [DiaAdlhKkk] --> [Kata] --> NULL
                                    tail --> [Kata]

Add Last

Menambahkan node baru di posisi paling belakang. Node baru menjadi tail baru. Operasi O(1).

Kode Program

C++ 
void addLast(string judul, string pengarang, int tB) {
    newNode = new Buku();
    newNode->judul       = judul;
    newNode->pengarang   = pengarang;
    newNode->tahunTerbit = tB;
    newNode->next = NULL;      // node baru adalah akhir list
    tail->next    = newNode;   // sambungkan dari tail lama
    tail          = newNode;   // geser tail ke node baru
}
Poin Penting:
  • newNode->next = NULL → node baru adalah ujung list
  • tail->next = newNode → sambungkan tail lama ke node baru
  • tail = newNode → geser tail ke node baru; head tidak berubah

Add Middle

Menambahkan node baru di posisi tertentu di tengah list. Membutuhkan traversal O(n).

Kode Program

C++ 
void addMiddle(string judul, string pengarang, int tB, int posisi) {
    if (posisi < 1 || posisi > countSingleLinkedList()) {
        cout << "Posisi diluar jangkauan" << endl;
    } else if (posisi == 1) {
        cout << "Posisi bukan posisi tengah" << endl;
    } else {
        newNode = new Buku();
        newNode->judul       = judul;
        newNode->pengarang   = pengarang;
        newNode->tahunTerbit = tB;

        cur = head;
        int nomor = 1;
        while (nomor < posisi - 1) {   // berhenti di posisi - 1
            cur = cur->next;
            nomor++;
        }
        newNode->next = cur->next;   // sambung ke node lama di posisi
        cur->next     = newNode;     // sambung dari node sebelum
    }
}
Dua Langkah Penyambungan:
  • newNode->next = cur->next → sambungkan dulu ke depan
  • cur->next = newNode → baru sambungkan dari belakang
  • Urutan ini WAJIB; terbalik akan kehilangan referensi ke node setelah posisi sisipan

Visualisasi

List: [A] --> [B] --> [C]
Sisip [X] di posisi 2

Traversal: berhenti di posisi-1=1, cur = [A]
newNode->next = cur->next = [B]    --> [X] --> [B]
cur->next     = newNode             --> [A] --> [X]

Sesudah: [A] --> [X] --> [B] --> [C]

Remove First

Menghapus node pertama (head). Head bergeser ke node kedua. Operasi O(1).

Kode Program

C++ 
void removeFirst() {
    del  = head;         // simpan pointer head lama
    head = head->next;   // head bergeser ke node kedua
    delete del;          // bebaskan memori head lama
}
Poin Penting:
  • del = head → simpan dulu sebelum head digeser
  • head = head->next → head baru adalah node kedua
  • delete del → tanpa ini terjadi memory leak

Remove Last

Menghapus node terakhir (tail). Harus traversal O(n) karena tidak ada pointer prev.

Kode Program

C++ 
void removeLast() {
    del = tail;               // simpan tail yang akan dihapus
    cur = head;
    while (cur->next != tail) {   // traversal: cari node sebelum tail
        cur = cur->next;
    }
    tail       = cur;         // node sebelum tail menjadi tail baru
    tail->next = NULL;        // tail baru tidak punya penerus
    delete del;               // bebaskan memori tail lama
}
⚠ Mengapa Perlu Traversal?
  • SLL tidak punya pointer prev — tidak bisa langsung tahu node sebelum tail
  • Solusi: traversal dari head, berhenti saat cur->next == tail
  • Berbeda dari DLL yang bisa langsung akses tail->prev → O(1)

Remove Middle

Menghapus node di posisi tertentu di tengah list.

Kode Program

C++ 
void removeMiddle(int posisi) {
    if (posisi < 1 || posisi > countSingleLinkedList()) {
        cout << "Posisi diluar jangkauan" << endl;
    } else if (posisi == 1) {
        cout << "Posisi bukan posisi tengah" << endl;
    } else {
        int nomor = 1;
        cur = head;
        while (nomor <= posisi) {
            if (nomor == posisi - 1) before = cur;   // simpan node sebelum target
            if (nomor == posisi)     del    = cur;   // simpan node target
            cur = cur->next;
            nomor++;
        }
        before->next = cur;   // bypass node target
        delete del;
    }
}
Teknik Bypass dengan Dua Pointer:
  • before → node di posisi target - 1
  • del → node di posisi target
  • Setelah loop, cur sudah maju satu langkah setelah del
  • before->next = cur → sambungkan langsung, melewati del

Apa itu Double Linked List?

Double Linked List (DLL) adalah linked list di mana setiap node memiliki dua pointer: prev (ke node sebelumnya) dan next (ke node berikutnya). Ini memungkinkan traversal dua arah dan operasi hapus di akhir menjadi O(1).

Keunggulan DLL vs SLL:
  • removeLast O(1) → tinggal akses tail->prev; SLL butuh O(n)
  • Traversal dua arah → bisa maju dan mundur
  • Trade-off → setiap node butuh memori extra untuk pointer prev
  • Cocok untuk: browser history, undo/redo, music player
NULL <-- [prev|data|next] <--> [prev|data|next] <--> [prev|data|next] --> NULL
          head                                          tail

Deklarasi Struct & Variabel Global

C++ 
struct DataUser {
    string nama, username, email, password;
    DataUser *prev;   // pointer ke node sebelumnya; NULL jika node pertama
    DataUser *next;   // pointer ke node berikutnya; NULL jika node terakhir
};

DataUser *head;
DataUser *tail;
DataUser *cur;
DataUser *newNode;
DataUser *del;
DataUser *afterNode;  // node setelah posisi sisipan (untuk addMiddle)
Perbedaan Struct DLL vs SLL:
  • DLL punya dua pointer per node: prev dan next
  • prev di head selalu NULL; next di tail selalu NULL
  • Saat insert di tengah, baik prev maupun next dari node sekitar harus diperbarui

Create

C++ 
void createDoubleLinkedList(string data[4]) {
    head = new DataUser();
    head->nama     = data[0];
    head->username = data[1];
    head->email    = data[2];
    head->password = data[3];
    head->prev = NULL;   // node pertama tidak punya pendahulu
    head->next = NULL;   // node pertama tidak punya penerus
    tail = head;
}
Perbedaan Create DLL vs SLL:
  • DLL harus mengisi dua pointer: prev = NULL dan next = NULL
  • SLL hanya perlu next = NULL

Add First

Menambahkan node baru di depan. Pointer prev dari head lama juga harus disambungkan ke node baru.

C++ 
void addFirst(string data[4]) {

Add Middle

Pada DLL ada 4 pointer yang harus diperbarui karena setiap sambungan bersifat dua arah.

C++ 
void addMiddle(string data[4], int posisi) {
    if (head == NULL) {
        cout << "Double Linked List belum dibuat!!!" << endl;
    } else {
        if (posisi == 1) {
            cout << "Posisi 1 itu bukan posisi tengah!!!" << endl;
        } else if (posisi < 1 || posisi > countDoubleLinkedList()) {
            cout << "Posisi diluar jangkauan!!!" << endl;
        } else {
            newNode = new DataUser();
            newNode->nama     = data[0];
            newNode->username = data[1];
            newNode->email    = data[2];
            newNode->password = data[3];

            cur = head;
            int nomor = 1;
            while (nomor < posisi - 1) {
                cur = cur->next;
                nomor++;
            }
            afterNode = cur->next;
            newNode->prev   = cur;        // sambung ke kiri
            newNode->next   = afterNode;  // sambung ke kanan
            cur->next       = newNode;    // node kiri ke newNode
            afterNode->prev = newNode;    // node kanan ke newNode
        }
    }
}
4 Pointer yang Harus Diperbarui:
  • newNode->prev = cur
  • newNode->next = afterNode
  • cur->next = newNode
  • afterNode->prev = newNode

Remove First

Menghapus node pertama. prev dari head baru harus diset ke NULL.

C++ 
void removeFirst() {
    if (head == NULL) {
        cout << "Double Linked List belum dibuat!!!" << endl;
    } else {
        del  = head;
        head = head->next;
        head->prev = NULL;   // head baru tidak punya pendahulu
        delete del;
    }
}
Perbedaan removeFirst DLL vs SLL:
  • SLL: cukup head = head->next
  • DLL: setelah head digeser, wajib head->prev = NULL

Remove Last

Karena DLL punya pointer prev, operasi ini O(1) — tidak perlu traversal.

C++ 
void removeLast() {
    if (head == NULL) {
        cout << "Double Linked List belum dibuat!!!" << endl;
    } else {
        del  = tail;
        tail = tail->prev;   // tail baru = node sebelum tail lama (pakai prev!)
        tail->next = NULL;
        delete del;
    }
}
Keunggulan DLL — removeLast O(1):
  • SLL removeLast: harus traversal O(n) dari head
  • DLL removeLast: cukup tail = tail->prev → O(1)

Remove Middle

C++ 
void removeMiddle(int posisi) {
    if (head == NULL) {
        cout << "Double Linked List belum dibuat!!!" << endl;
    } else {
        if (posisi == 1 || posisi == countDoubleLinkedList()) {
            cout << "Posisi bukan posisi tengah!!" << endl;
        } else if (posisi < 1 || posisi > countDoubleLinkedList()) {
            cout << "Posisi diluar jangkauan!!" << endl;
        } else {
            int nomor = 1;
            cur = head;
            while (nomor < posisi - 1) {
                cur = cur->next;
                nomor++;
            }
            del       = cur->next;
            afterNode = del->next;
            cur->next       = afterNode;  // node kiri skip del
            afterNode->prev = cur;        // node kanan skip del
            delete del;
        }
    }
}
2 Pointer yang Diperbarui saat Remove Middle DLL:
  • cur->next = afterNode → node kiri langsung menunjuk ke node kanan
  • afterNode->prev = cur → node kanan balik menunjuk ke node kiri

Apa itu Circular Single Linked List?

Circular Single Linked List (CSLL) adalah Single Linked List di mana tail->next tidak menunjuk ke NULL, melainkan menunjuk kembali ke head. Hasilnya adalah struktur melingkar — tidak ada ujung yang NULL.

Perbedaan CSLL vs SLL:
  • SLL: tail->next = NULL
  • CSLL: tail->next = head → membentuk lingkaran
  • Traversal: gunakan while(cur->next != head), bukan while(cur != NULL)
  • Cocok untuk: round-robin scheduling, playlist berulang, game giliran pemain
+-------------------------------------+
|                                     |
+-> [A] -> [B] -> [C] -> [D] --------+
    head                    tail

Deklarasi Struct & Variabel Global

C++ 
struct Mahasiswa {
    string npm, nama, jurusan;
    Mahasiswa *next;   // tidak pernah NULL — selalu menunjuk ke node berikutnya
};

Mahasiswa *head;      // node pertama
Mahasiswa *tail;      // node terakhir; tail->next == head selalu
Mahasiswa *newNode;
Mahasiswa *cur;
Mahasiswa *del;
⚠ Invariant CSLL yang Harus Selalu Terjaga:
  • tail->next == head → SELALU, sebelum dan sesudah setiap operasi
  • Jika invariant ini rusak, traversal akan infinite loop atau crash

Create

C++ 
void createCircularSingleLinkedList(string dataBaru[3]) {
    head = new Mahasiswa();
    head->npm     = dataBaru[0];
    head->nama    = dataBaru[1];
    head->jurusan = dataBaru[2];
    tail = head;
    tail->next = head;   // KUNCI: node satu-satunya menunjuk ke dirinya sendiri
}
Perbedaan Create CSLL vs SLL:
  • SLL: head->next = NULL
  • CSLL: tail->next = head → sirkular sejak node pertama

Add First

C++ 
void addFirst(string data[3]) {
    if (head == NULL) {
        cout << "Buat Linked List dulu!!" << endl;
    } else {
        newNode = new Mahasiswa();
        newNode->npm     = data[0];
        newNode->nama    = data[1];
        newNode->jurusan = data[2];
        newNode->next = head;    // node baru menunjuk ke head lama
        tail->next    = newNode; // tail harus menunjuk ke head baru!
        head          = newNode; // geser head
    }
}
⚠ Beda dengan SLL addFirst:
  • SLL: 2 langkah — newNode->next = head, head = newNode
  • CSLL: 3 langkah — tambah tail->next = newNode agar lingkaran tidak putus

Add Last

C++ 
void addLast(string data[3]) {

Beda CSLL addLast vs SLL addLast:
  • SLL: newNode->next = NULL
  • CSLL: newNode->next = head → node baru langsung menutup lingkaran

Add Middle

C++ 
void addMiddle(string data[3], int posisi) {
    if (head == NULL) {
        cout << "Buat Linked List dulu!!" << endl;
    } else {
        if (posisi == 1) {
            cout << "Posisi satu bukan posisi tengah!!!" << endl;
        } else {
            newNode = new Mahasiswa();
            newNode->npm     = data[0];
            newNode->nama    = data[1];
            newNode->jurusan = data[2];

            int nomor = 1;
            cur = head;
            while (nomor < posisi - 1) {
                cur = cur->next;
                nomor++;
            }
            newNode->next = cur->next;
            cur->next     = newNode;
        }
    }
}
Catatan addMiddle CSLL:
  • Jika node disisipkan di tengah, sirkularitas tidak berpengaruh — polanya sama persis dengan SLL
  • Pointer tail->next tidak perlu disentuh karena head dan tail tidak berubah

Remove First

C++ 
void removeFirst() {
    if (head == NULL) {
        cout << "Buat Linked List dulu!!" << endl;
    } else {
        del  = head;
        head = head->next;
        tail->next = head;   // tail harus menunjuk ke head baru!
        delete del;
    }
}
Beda CSLL removeFirst vs SLL removeFirst:
  • SLL: cukup del = head; head = head->next; delete del
  • CSLL: wajib tambah tail->next = head agar tail tidak menunjuk ke node yang sudah dihapus

Remove Last

C++ 
void removeLast() {
    if (head == NULL) {
        cout << "Buat Linked List dulu!!" << endl;
    } else {
        del = tail;
        cur = head;
        while (cur->next != tail) {   // berhenti di node sebelum tail
            cur = cur->next;
        }
        tail       = cur;
        tail->next = head;   // tail baru menunjuk ke head (bukan NULL!)
        delete del;
    }
}
Beda CSLL removeLast vs SLL removeLast:
  • SLL: tail->next = NULL setelah tail digeser
  • CSLL: tail->next = head — tail baru harus tutup lingkaran, bukan NULL

Remove Middle

Catatan removeMiddle CSLL:
  • Jika node yang dihapus bukan head maupun tail, pointer tail->next tidak terpengaruh
  • Polanya identik dengan SLL removeMiddle

Kesimpulan — Perbandingan Tiga Linked List

Perbedaan Kunci Tiga Jenis Linked List:
  • Struct → SLL & CSLL: hanya next. DLL: prev + next
  • Create → SLL: next=NULL. DLL: prev=NULL, next=NULL. CSLL: next=head
  • addFirst → SLL: 2 langkah. DLL: 4 langkah (update head->prev). CSLL: 3 langkah (update tail->next)
  • addLast → SLL: newNode->next=NULL. DLL: newNode->prev=tail. CSLL: newNode->next=head
  • addMiddle → SLL & CSLL: 2 pointer. DLL: 4 pointer
  • removeFirst → SLL: biasa. DLL: tambah head->prev=NULL. CSLL: tambah tail->next=head
  • removeLast → SLL: O(n) traversal. DLL: O(1) via tail->prev. CSLL: O(n) + tail->next=head
  • Traversal kondisi berhenti → SLL & DLL: cur != NULL. CSLL: cur->next != head
