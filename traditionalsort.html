1. Bubble Sort

Bubble Sort bekerja dengan membandingkan dua elemen yang berdekatan dan menukarnya jika urutannya salah. Elemen terbesar akan "mengapung" ke posisi akhir seperti gelembung. Terdapat optimasi dengan flag swapped agar berhenti lebih awal jika array sudah tersortir.

Kode Program

C++ 
void bubbleSort(int arr[], int length)
{
    for (int i = 0; i < length - 1; i++)
    {
        bool swapped = false;

        // Bandingkan pasangan elemen berdekatan
        for (int j = 0; j < length - i - 1; j++)
        {
            if (arr[j] > arr[j + 1])
            {
                swap(arr[j], arr[j + 1]);
                swapped = true;
            }
        }

        // Jika tidak ada pertukaran, array sudah tersortir
        if (!swapped)
            break;
    }
}

Penjelasan Program

Cara Kerja Bubble Sort:
  • Loop luar (i) menentukan berapa elemen di akhir yang sudah tersortir
  • Loop dalam (j) membandingkan arr[j] dengan arr[j+1]
  • Jika arr[j] > arr[j+1], lakukan swap → elemen besar "gelembung" ke kanan
  • Setiap iterasi loop luar, elemen terbesar sudah berada di posisi akhir yang benar
  • Flag swapped: jika satu putaran penuh tidak ada swap, array sudah terurut → berhenti
⏱ Kompleksitas:
  • Best Case: O(n) — jika array sudah terurut (berkat flag swapped)
  • Average & Worst Case: O(n²)
  • Jumlah swap bisa sangat banyak dibanding Selection Sort

2. Selection Sort

Selection Sort bekerja dengan cara mencari elemen terkecil dari bagian array yang belum tersortir, lalu menukarnya ke posisi paling depan. Prosesnya berulang hingga seluruh array tersusun secara ascending.

Kode Program

C++ 
void selectionSort(int arr[], int length)
{
    int minIndex;
    for (int i = 0; i < length - 1; i++)
    {
        minIndex = i;

        // Cari index elemen terkecil di sisa array
        for (int j = i + 1; j < length; j++)
        {
            if (arr[minIndex] > arr[j])
            {
                minIndex = j;
            }
        }

        // Tukar elemen terkecil ke posisi i
        swap(arr[i], arr[minIndex]);
    }
}

Penjelasan Program

Cara Kerja Selection Sort:
  • Loop luar (i) menentukan posisi yang akan diisi — mulai dari indeks 0
  • minIndex = i → anggap elemen posisi i adalah yang terkecil dulu
  • Loop dalam (j) mencari elemen yang lebih kecil dari arr[minIndex]
  • Jika ditemukan yang lebih kecil, perbarui minIndex
  • Setelah loop dalam selesai, tukar arr[i] dengan arr[minIndex]
  • Posisi ke-i kini sudah berisi elemen yang benar dan tidak akan digeser lagi
⏱ Kompleksitas:
  • Best, Average, Worst Case: O(n²) — selalu dua loop bersarang
  • Jumlah perbandingan selalu sama terlepas dari kondisi awal array
  • Jumlah swap maksimal hanya n-1 kali — lebih sedikit dari Bubble Sort

3. Insertion Sort

Insertion Sort bekerja seperti menyusun kartu di tangan — ambil satu kartu, lalu sisipkan ke posisi yang tepat di antara kartu-kartu yang sudah tersusun. Array terbagi menjadi bagian tersortir (kiri) dan belum tersortir (kanan).

Kode Program

C++ 
void insertionSort(int arr[], int length)
{
    for (int i = 1; i < length; i++)
    {
        int key = arr[i];   // Elemen yang akan disisipkan
        int j = i - 1;

        // Geser elemen yang lebih besar dari key ke kanan
        while (j >= 0 && arr[j] > key)
        {
            arr[j + 1] = arr[j];
            j--;
        }

        // Sisipkan key ke posisi yang benar
        arr[j + 1] = key;
    }
}

Penjelasan Program

Cara Kerja Insertion Sort:
  • Loop dimulai dari indeks 1 — elemen pertama dianggap sudah tersortir sendiri
  • key = arr[i] → simpan elemen yang akan disisipkan
  • while: selama elemen di kiri lebih besar dari key, geser ke kanan
  • Setelah loop while, posisi j+1 adalah tempat yang tepat untuk key
  • Sisipkan key ke posisi tersebut
⏱ Kompleksitas:
  • Best Case: O(n) — jika array sudah hampir terurut, loop while hampir tidak berjalan
  • Average & Worst Case: O(n²)
  • Paling efisien untuk array kecil atau array yang hampir tersortir

4. Kode Lengkap

Berikut kode lengkap ketiga algoritma digabung dalam satu program dengan fungsi bantu printChanges dan display.

Kode Program

C++ 
#include <iostream>
using namespace std;

void printChanges(int arr[], int length, int step)
{
    cout << "Langkah ke-" << step << ": [ ";
    for (int i = 0; i < length; i++)
    {
        cout << arr[i];
        if (i < length - 1) cout << ", ";
    }
    cout << " ]" << endl;
}

void display(int arr[], int length)
{
    cout << "Hasil akhir: [ ";
    for (int i = 0; i < length; i++)
    {
        cout << arr[i];
        if (i < length - 1) cout << ", ";
    }
    cout << " ]" << endl;
    cout << "==========================================" << endl;
}

void selectionSort(int arr[], int length)
{
    int minIndex;
    for (int i = 0; i < length - 1; i++)
    {
        minIndex = i;
        for (int j = i + 1; j < length; j++)
            if (arr[minIndex] > arr[j]) minIndex = j;
        swap(arr[i], arr[minIndex]);
        printChanges(arr, length, i + 1);
    }
}

void bubbleSort(int arr[], int length)
{
    for (int i = 0; i < length - 1; i++)
    {
        bool swapped = false;
        for (int j = 0; j < length - i - 1; j++)
        {
            if (arr[j] > arr[j + 1])
            {
                swap(arr[j], arr[j + 1]);
                swapped = true;
            }
        }
        printChanges(arr, length, i + 1);
        if (!swapped) break;
    }
}

void insertionSort(int arr[], int length)
{
    for (int i = 1; i < length; i++)
    {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key)
        {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
        printChanges(arr, length, i);
    }
}

int main()
{
    int arr[]  = {7, 3, 5, 9, 2, 1, 8};
    int arr2[] = {7, 3, 5, 9, 2, 1, 8};
    int arr3[] = {7, 3, 5, 9, 2, 1, 8};
    int length = sizeof(arr) / sizeof(int);

    cout << "Proses Selection Sort:" << endl;
    selectionSort(arr, length);
    display(arr, length);

    cout << "\nProses Bubble Sort:" << endl;
    bubbleSort(arr2, length);
    display(arr2, length);

    cout << "\nProses Insertion Sort:" << endl;
    insertionSort(arr3, length);
    display(arr3, length);

    return 0;
}

5. Perbandingan Ketiga Algoritma

Ringkasan Perbandingan:
  • Selection Sort → swap paling sedikit (n-1 kali), cocok jika operasi swap mahal. Selalu O(n²).
  • Bubble Sort → paling mudah dipahami, bisa berhenti lebih awal (best case O(n)), tapi swap terbanyak.
  • Insertion Sort → sangat efisien untuk data hampir terurut, best case O(n), stabil dan praktis untuk array kecil.
💡 Kapan Pakai yang Mana?
  • Data kecil (< 20 elemen) → ketiganya setara, pilih yang paling mudah dibaca
  • Data hampir tersortir → Insertion Sort paling cepat
  • Ingin minimasi swap → Selection Sort
  • Data besar → lebih baik pakai Quick Sort atau Merge Sort (O(n log n))

Kesimpulan

Ketiga algoritma traditional sort memiliki kompleksitas O(n²) di kasus rata-rata dan terburuk, sehingga kurang cocok untuk data dalam jumlah besar. Namun, memahaminya sangat penting sebagai fondasi berpikir algoritmik sebelum mempelajari algoritma yang lebih canggih.

Poin-Poin Penting:
  • Selection Sort: Cari minimum → tukar ke posisi benar. Swap paling hemat.
  • Bubble Sort: Bandingkan tetangga → gelembungkan yang besar. Ada optimasi early exit.
  • Insertion Sort: Ambil satu elemen → sisipkan ke posisi yang tepat di bagian tersortir.
  • Semua menggunakan dua loop bersarang → O(n²) average case.
  • Bubble & Insertion Sort bisa O(n) di best case jika data hampir terurut.
  • Selection Sort tidak stabil; Bubble & Insertion Sort stabil.
